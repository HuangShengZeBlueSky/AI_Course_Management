# Contributing / 协作方式

这个仓库用于课程“项目化”管理。

## 学员提问

- 用 Issue 提问：标题写清楚（模块 + 问题），内容包含：复现步骤、报错信息、代码片段/截图。
- 如问题涉及隐私/账号信息，请不要直接贴出来。

## 作业提交

- 每次作业用 PR 提交到对应模块目录：
  - `assignments/` 放题目与验收标准
  - `projects/<模块>/` 放项目代码与说明
- PR 描述写：完成了什么、怎么跑、结果截图/指标。

## 代码与文档约定（建议）

- 每个项目目录至少包含一个 `README.md`（如何安装/运行/复现）。
- 依赖写在 `requirements.txt` 或 `environment.yml`（后续需要我可以补上模板）。
