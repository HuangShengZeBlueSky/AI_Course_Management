# 基础知识（Fundamentals）模块

这个模块用于把后续所有项目需要的“共同基础”补齐。

## 覆盖内容

- AI 基本概念：数据/特征、训练/验证/测试、过拟合、常用指标
- Python 基础：语法、函数、类、异常、工程化习惯
- 工具链：环境管理（venv/conda）、VS Code、Git/GitHub 协作

## 产出

- 一套可复现的开发环境
- 一套可运行的基础示例代码（后续项目直接复用）

## 里程碑（建议）

- [ ] 环境与依赖可复现（给出安装与运行命令）
- [ ] Python 基础练习（含答案或讲解）
- [ ] GitHub 协作演练（Issue/PR/Review）
