# 大模型 LLM 模块

这个模块聚焦“如何正确使用与构建 LLM 应用”，并为 RAG/Agent 项目打地基。

## 覆盖内容

- LLM 基础：token、上下文、温度、系统/用户提示
- 提示工程：任务约束、输出格式、示例驱动、可靠性校验
- 评估与安全：幻觉、可追溯、成本与延迟、敏感信息处理
- 实战方向：
  - 文本任务（摘要/分类/抽取）
  - 工具调用（function calling / tool use）
  - 为 RAG/Agent 做准备

- [ ] 基础 Prompt 模板与输出规范（JSON/Markdown 等）
- [ ] 可靠性策略（重试/校验/约束解码/工具回读）
- [ ] 小型 Demo（一个可展示的 LLM 应用）
